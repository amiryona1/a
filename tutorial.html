Tutorial

This tutorial will guide you through the basic usage of Terminal-Colorify. Follow the steps below to get started quickly!

  1. Install Terminal-Colorify with pip: 
    pip install terminal-colorify
  2. Import the library into your Python project: 
    from colors import Colors
  3. Print colored text with simple functions: 
    print(Colors.set_color(color="green", text="Operation Successful"))
  4. Apply gradients or rainbow effects: 
    print(Colors.gradient_text("Gradient Example", (255,0,0), (0,0,255)))
print(Colors.rainbow_text("Rainbow Example"))
  5. Use built-in themes for quick styling: 
    print(Colors.theme_text("Warning message", theme="warning"))

That's it! Youre ready to make your terminal colorful and dynamic. More advanced features will be available in the full documentation soon.
